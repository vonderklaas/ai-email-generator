# AI Email Test Prompts

Use these prompts to judge whether the generator produces emails a SaaS founder would actually ship.

## Core Set

1. Trial ending in 3 days for a $49/month design feedback tool. Offer 20% off annual if they upgrade before Friday.
2. Welcome email for a new free user of a team wiki app. Goal: get them to invite one teammate.
3. Feature announcement for a project management SaaS launching native Slack approvals.
4. Re-engagement email for users who signed up for an AI meeting notes app but never connected their calendar.
5. Win-back email for a churned customer of a customer support analytics tool. New feature: AI ticket tagging.
6. Migration notice for a developer API product changing webhook signing secrets next month.
7. Case study share for a B2B SaaS showing that a customer reduced onboarding time by 32%.
8. Trial extension offer for a startup CRM where the user created contacts but has not sent any emails.
9. Product update newsletter for a small finance dashboard announcing CSV import, saved reports, and faster exports.
10. Webinar invitation for a SaaS founder audience about improving activation with lifecycle emails.
11. Onboarding reminder for a password manager team admin who has not invited employees yet.
12. Pricing change announcement for an uptime monitoring tool moving from unlimited free checks to a 10-check free plan.
13. Usage milestone email for a forms product celebrating the user's first 100 responses.
14. Upgrade nudge for a design system documentation tool after the user hits the free project limit.
15. Apology email for an analytics SaaS after a 47-minute partial outage, with a clear status page CTA.

## Evaluation Rule

For each prompt, score the output with `docs/ai-email-quality-rubric.md`. A v0.1 prompt is acceptable when at least 70% of the core set is ship-ready without major manual edits.
